GoMining Referral Export

Exportiere alle deine GoMining Referral-Transaktionen als CSV. Direkt über die Browser-Console – vollautomatisch via API.

bolt

Vollautomatisch

Kein manuelles Copy & Paste. Alle Seiten automatisch durchlaufen.

date_range

Datumsfilter

Exportiere nach Jahr, Monat oder benutzerdefiniertem Zeitraum.

lock

Sicher & Lokal

Deine Daten bleiben im Browser. Kein Upload, keine externe Verarbeitung.

format_list_numbered Schritt-für-Schritt Anleitung

1

GoMining.com öffnen

Gehe zu app.gomining.com und logge dich ein.

2

Browser-Console öffnen

Drücke F12 oder Strg + Shift + J

Wechsle zum "Console" Tab.

💡 Tipp: Klicke einmal in die Console, damit sie fokussiert ist.

3

Skript laden

Kopiere das komplette Skript und füge es in die Console ein:

code Skript öffnen 
// Das komplette Skript findest du hier:
// src/scripts/console_scripts/gomining-referral-complete-export_approved.js

// Oder direkt laden:
fetch('../src/scripts/console_scripts/gomining-referral-complete-export_approved.js')
  .then(r => r.text())
  .then(code => eval(code));
4

Export starten

Wähle einen Export-Befehl:

📊 Alle Daten (empfohlen)

exportAllReferralData()

Holt alle Referral-Transaktionen seit Beginn

📅 Ganzes Jahr

exportReferralDataRange("2026-01-01", "2026-12-31")

Alle Referral-Transaktionen von 2026

📆 Einzelner Monat

exportReferralDataRange("2026-04-01", "2026-04-30")  // April 2026

Beliebiger Zeitraum (Format: YYYY-MM-DD)

💾 CSV herunterladen

exportTransactionsToCSV()

Nach dem Laden: CSV-Datei herunterladen

🛑 Export abbrechen

stopExport()

Laufenden Export stoppen (dauert wenige Sekunden)

info Wichtig: 2-Schritt-Prozess

  1. 1
    Daten laden: Führe exportAllReferralData() oder exportReferralDataRange(...) aus. Die Daten werden heruntergeladen und im Speicher gehalten.
  2. 2
    CSV exportieren: Führe danach exportTransactionsToCSV() aus, um die CSV-Datei herunterzuladen.

help_outline Häufige Fragen

Wie finde ich meinen Token?

Das Skript sucht den Token automatisch in Cookies und localStorage. Falls es nicht klappt: DevTools → Network Tab → einen API-Request anklicken → "Authorization" Header kopieren. Dann globalThis.goMiningToken = "TOKEN" setzen.

Ist das sicher?

Ja! Das Skript läuft vollständig lokal in deinem Browser. Es sendet keine Daten an externe Server. Du kannst den Quellcode jederzeit einsehen und prüfen.

Der Export läuft sehr lange – warum?

Die Referral-API liefert nur 20 Einträge pro Anfrage. Bei vielen Transaktionen (z.B. 50.000+) dauert der Export entsprechend länger. Mit Datumsfilter (exportReferralDataRange) lässt sich die Datenmenge deutlich reduzieren.

Warum bekomme ich einen 401 Fehler?

Dein Token ist abgelaufen. Lade die Seite neu (F5), warte bis sie vollständig geladen ist, und führe das Skript erneut aus.

Wie öffne ich die CSV in Excel?

In Excel: Datei → Öffnen → "Alle Dateien" → CSV auswählen → Im Import-Dialog: Trennzeichen "Semikolon" und Dezimaltrennzeichen "Komma" wählen.
