help_outline Häufige Fragen

Was ist der Unterschied zur alten miner-history-import.html?

Die alte Version erfordert manuelles Copy & Paste aus dem GoMining UI. Diese neue Version holt alle Daten vollautomatisch direkt über die API - schneller, einfacher, zuverlässiger.

Werden wirklich alle Daten exportiert?

Ja! Das Skript durchläuft automatisch alle Seiten der API-Pagination und holt jeden einzelnen Eintrag. Du bekommst deine komplette Historie ohne Lücken.

Wie finde ich meinen Token?

Das Skript findet den Token automatisch aus Cookies oder localStorage. Falls nicht, öffne DevTools → Network Tab → Suche nach API-Requests → Kopiere den "Authorization" Header.

Kann ich die CSVs in Excel öffnen?

Ja! In Excel: Datei → Öffnen → Wähle "Alle Dateien" → CSV auswählen → Im Import-Dialog: Trennzeichen "Semikolon" wählen.

Wie lange dauert der Export?

Das hängt von der Anzahl deiner Miner ab:
• ~10 Miner: 2-5 Sekunden
• ~100 Miner: 10-20 Sekunden
• ~500 Miner: 30-60 Sekunden
