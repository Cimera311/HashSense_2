GoMining Transaction Export

Exportiere alle deine GoMining Transaktionen in Sekunden als CSV-Datei. Schnell, einfach und direkt über die Browser-Console.

list_alt Schritt-für-Schritt Anleitung

1

GoMining.com öffnen

Gehe zu app.gomining.com und logge dich ein.

2

Browser-Console öffnen

Drücke F12 oder Strg + Shift + J

Wechsle zum "Console" Tab.

💡 Tipp: Klicke einmal in die Console, damit sie fokussiert ist.

3

Skript laden

Kopiere das komplette Skript und füge es in die Console ein:

code Skript öffnen 
// Das komplette Skript findest du hier:
// src/scripts/console scripts/gomining-export-final.js

// Oder direkt laden:
fetch('src/scripts/console scripts/gomining-export-final.js')
  .then(r => r.text())
  .then(code => eval(code));
4

Export starten

Wähle einen Export-Befehl:

🧪 Test (empfohlen zuerst!)

testExport()

Lädt nur 100 Transaktionen zum Testen

📅 Ganzes Jahr

exportYear(2026)

Alle Transaktionen von 2026

📆 Einzelner Monat

exportMonth(2026, 4)  // April 2026

2. Parameter ist der Monat (1-12)

🎯 Benutzerdefiniert

exportFromTo("2026-01-01", "2026-04-02")

Beliebiger Zeitraum (Format: YYYY-MM-DD)

⚠️ Alle Transaktionen

exportAll()

❗ Kann lange dauern bei vielen Transaktionen!

settings Erweiterte Optionen

Datumsbereich manuell setzen

setDateRange("2026-01-01", "2026-12-31")
exportTransactions()

Shortcuts

setDateRange("jahr")   // Dieses Jahr
setDateRange("monat")  // Dieser Monat
setDateRange("heute")  // Nur heute
setDateRange(2026)     // Jahr als Zahl

Konfiguration anpassen

exportConfig.batchSize = 200  // Mehr pro Request

Token manuell setzen

globalThis.goMiningToken = "DEIN_TOKEN"
exportTransactions()

help_outline Häufige Fragen

Wie finde ich meinen Token?

Das Skript findet den Token automatisch aus Cookies oder localStorage. Falls nicht, öffne DevTools → Network Tab → Suche nach "transaction-history" → Kopiere den "Authorization" Header.

Ist das sicher?

Ja! Das Skript läuft komplett lokal in deinem Browser. Es sendet keine Daten an externe Server. Du kannst den Code jederzeit einsehen und prüfen.

Warum bekomme ich einen 401 Fehler?

Dein Token ist abgelaufen. Lade die Seite neu (F5), warte bis sie vollständig geladen ist, und führe das Skript erneut aus. Bei Bedarf kannst du den Token auch manuell aus DevTools kopieren.

Wie öffne ich die CSV in Excel?

In Excel: Datei → Öffnen → Wähle "Alle Dateien" → CSV auswählen → Im Import-Dialog: Trennzeichen "Semikolon" und Dezimaltrennzeichen "Komma" wählen.

Wie lange dauert der Export?

Das hängt von der Anzahl der Transaktionen ab:
• ~1.000 Transaktionen: 5-10 Sekunden
• ~10.000 Transaktionen: 30-60 Sekunden
• ~68.000 Transaktionen: 1-2 Minuten
