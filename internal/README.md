# Internal Bereich

Dieser Ordner ist fuer Dateien, die im GitHub-Repository liegen sollen,
aber nicht ueber GitHub Pages (hashfarm.me) veroeffentlicht werden.

Wichtig:
- In einem oeffentlichen Repo bleiben diese Dateien weiterhin auf GitHub sichtbar.
- Damit sie nicht ueber hashfarm.me erreichbar sind, muss GitHub Pages auf einen
  anderen Source-Bereich zeigen (z. B. Branch `gh-pages` oder Ordner `docs/`).
